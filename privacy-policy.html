EXPLICIT STATEMENT:

Face data is NOT retained by HAIRGATOR or any third party.

Face Shape Analysis & Personal Color Analysis:
These features use MediaPipe (Google's on-device ML library) which runs entirely on the user's device (browser/app). Face data is never transmitted to HAIRGATOR servers or any third party. All facial analysis is performed locally on the user's device and no face data leaves the device.

Virtual Hairstyle Try-On:
This feature uses vModel API. The user's photo is sent to vModel's server only for processing and is deleted immediately after the result image is generated. vModel does not store, retain, or use the uploaded images for any other purpose including AI training.

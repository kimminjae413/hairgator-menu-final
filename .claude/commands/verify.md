# 자기 검증 모드

방금 내가 한 주장들을 다시 검토하세요.

## 검토 항목

1. **각 주장의 확신도 분류**
   - 🟢 확실함: 직접 코드를 읽거나 테스트로 확인함
   - 🟡 추측: 문서나 경험에 기반한 추측
   - 🔴 모름: 확인이 필요함

2. **추측/모름인 것들**
   - 어떻게 확인할 수 있는지 구체적으로 제시
   - 확인 명령어가 있으면 **실제로 실행**

3. **에코체임버 체크**
   - 사용자가 듣고 싶어하는 말만 하고 있지 않나?
   - 불편하더라도 정확한 정보를 전달하고 있나?

## 출력 형식

```
### 주장 1: [주장 내용]
- 확신도: 🟢/🟡/🔴
- 근거: [확인한 방법 또는 추측 이유]
- 검증 필요시: [검증 방법]

### 주장 2: ...
```

**솔직하게 답변하세요. 틀렸을 가능성이 있으면 인정하세요.**
